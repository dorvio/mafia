# Mafia

## Opis

Aplikacja na wzór gry towarzyskiej Mafia. Gra przejmuje rolę Mistrza gry, co pozwala na aktywny udział w rozgrywce wszystkim uczestnikom.
Aplikacja przydziela graczom role, obsługuje głosowania w dzień oraz nocne głosowania zależne od roli graczy. Nadzoruje całą rozgrywkę, przelicza wyniki głososowań decyduje o końcu gry.

## Technologie

* Flutter – główny framework aplikacji.

* Dart – język programowania.

* Supabase Flutter – baza danych i backend jako usługa
  
* Shared Preferences – przechowywanie danych lokalnych
